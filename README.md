# fancyDownload
花式下载器，Linux下多线程下载工具，python多线程下载，  "IDM" on linux

# 依赖
sudo pip install -r requirements.txt

# 帮助
```bash
python fancyDownload.py $fileUrl $filePath

# eg: python fancyDownload.py https://xxx.com/1.zip ~/download/1.zip
# $fileurl: 文件链接
# $filePath: 完整存储路径
```
